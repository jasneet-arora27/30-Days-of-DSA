# 704. Binary Search

Difficulty: LeetCode Easy

```java
// This solution is in Java
class Solution {
    public int search(int[] nums, int target) {
        int s = 0, e = nums.length - 1;

        while (s <= e) {
            int mid = s + (e - s) / 2;
            if (nums[mid] == target) return mid;
            if (nums[mid] < target) {
                s = mid + 1;
            }
            else {
                e = mid - 1;
            }
        }

        return -1;
    }
}
```
